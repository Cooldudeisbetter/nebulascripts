# SPECTRA HUB

> Spectra is a wide variety of open-source programs which execute specific commands in Roblox games

## 🚀 Quick Start
```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/Cooldudeisbetter/nebulascripts/main/hub.lua"))()
```

## 💪 Advantages of Spectra

1. **Fast**
2. **Reliable**
3. **Effective**
